# Zakip Voice — Features

## Free (Open Source)

Everything in the core app is free and open source:

| Feature | Description |
|---------|-------------|
| **Voice transcription** | Record audio → transcribe via Whisper-compatible API |
| **AI text enhancement** | Polish transcribed text with any LLM |
| **Multiple providers** | OpenAI, Groq, Perplexity, OpenRouter, custom endpoints |
| **Global shortcuts** | System-wide keyboard shortcuts for recording |
| **Auto-paste** | Automatically paste transcription into focused app |
| **Clipboard copy** | One-click copy of transcription results |
| **Transcription history** | Browse, search, and replay past transcriptions |
| **Sound notifications** | Audible feedback on recording events |
| **System tray** | Background operation with tray icon |
| **Encrypted storage** | API keys stored with AES-GCM encryption |
| **Auto-update** | Built-in update mechanism |
| **Local Whisper** | On-device transcription (no API needed) |
| **Custom skills** | Create reusable AI workflows with voice input |

## Pro

Pro features require a license key. Available for early beta testers.

|---------|-------------|
